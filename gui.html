Click-GUI Generator

Generates a data-pack which has Click GUIs in it! Created by Exortions.
If you have any issues, feel free to contact me on Discord at Exo#0001. My DMs are open! :) Currently only works for 1.16, Not 1.17.

Run this command to summon a place for the GUI: <Can't be moven from the point its summoned without function <

execute as <summoner> at @s run function <namespace>:summon



Objectives & Tags







GUI Slots

Put in whatever you want to be put in a specific slot. e.g.

bone{Enchantments:[{id:"unbreaking",lvl:1}],HideFlags:31,display:{Name:'[{"text":"Colorful Bone!","color":"yellow"}]'},gui:1b}

Make sure to put a {gui:1b} in there somewhere or it wont work!































Click Events

Enter a command in each slot for what you want to happen when that slot is clicked. E.g:

tellraw @p {"text":"You clicked on me!","color":"red"}

Just make sure not to put a / in there!
